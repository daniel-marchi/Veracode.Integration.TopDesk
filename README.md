# Veracode.Integration.TopDesk

### Description
Script em Python para integração do Veracode SAST e SCA

### Getting Started

1.	Software dependencies
    - Python 3.9.16
    - Veracode
    - TopDesk
2.  Run
    - $ python3 ./vera2topdesk.py
